### Description

A button with an icon and/or label.

### Properties

| Property            | Type   |
| ------------------- | ------ |
| `ariaLabel`         | string |
| `attributes`        | array  |
| `iconClass`         | string |
| `label`             | string |

### Notes

`ariaLabel` is not used if there is a `label`.
